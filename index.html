HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Not all HTML is visible to the user
  • Alt text is important for accessibility
  • The text within the title tags will show up at the top of the browser tab

CSS

  • Your .css file must be linked to by your .html file
  • A margin is space outside of an element, padding is space around the content of an element
  • CSS can be added within an HTML file, but it's often preferrable to have an external CSS file
  • You can apply CSS properties to multiple elements by using element names or class attributes
  • You can use CSS to customize your colors, text, outlines, etc. however you like

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls all changes from main
  • git add -A: adds all modifications to the staging area
  • git commit -m "commit message here": describes what the commit contains
  • git push origin branchnamehere: pushes changes to the remote branch

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Javascript is a weakly typed language, and allows implicit type conversion
  • Make use of functions to avoid excessively repeating code
  • JavaScript is case sensitive
  • Use /* and */ to make a multi-line comment, or // if it's on one line
